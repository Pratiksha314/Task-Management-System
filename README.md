# Task-Management-System
A simple task management system built with mainly focusing on FastAPI and Angular


**Core Features**

- **User Authentication** – Register/login, JWT-based auth
- **Projects** – Create projects.
- **Tasks** – Create/edit tasks within a project, assign to users, set due dates, priority, status (To Do / In Progress / Done/ Backlog)
- **Comments** – Users can comment on tasks
- **Real-time-ish updates**


**Techinal Aspects**
(This Project mainly focus on Backend aspects to learn about FastAPI)

- Frontend : Angular (TypeScript)
- Backend : Fast API (Python)
- DB : PostgreSQL (SQLAlchemy)
